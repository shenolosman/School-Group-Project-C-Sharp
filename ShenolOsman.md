
# Projektuppgifts Rapporten

### Planering

Från början vi tänkte bygga programmet med ASP.net razor pages. Vi började lite med detta efter ett tag vi kunde inte ta den fram eftersom vi hade inte haft så många kunskap på detta.

Vi var 3 person och jobbade med olika branches så vi hade ibland konflikter ibland inte. Vi delade upp tre bitar. Jag byggde Admin sidan och andra tog andra sidor. Men ändå vi byggde tillsammans jag menar vi hjälpte varandra.


### Implementering

Vi har tagit backend sidan från gamla uppgiften och vi utvecklade den med frontend sidan med console app och med tester.

Frontend projekten har program.cs och det kallar 3 olika classer som byggdes för adminfrontend, restaurantfrontend och customerclient. Alla har sina egna menuer. I program.cs är huvud menu som kallar alla frontend classer. Varje gång databas recreateing om program startas om. 

Eftersom vi är tre byggde vi ändå login systemet och vi adderade admin och waitress users som styrt sida sidor.

Implementeringen med console app och backend var gick lätt att kommunicera var andra och vi hade inte träffat med stora problemet. Programmet funkar väl och vi försökte hindra att programmet crashar på något sätt genom att trycka på fel knappt eller något annat. 

Jag är nöjd med resultaten som vi gått snabbt till lösningen och byggde programet så snabbt. Menu blev jätte skönt och vi kunde också köra tester. Det är jag mest nöjd med.
